# Unithium Energy Systems LTD Website

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Getting Started
```bash
npm install
npm run dev
```

## Deployment
1. Push repo to GitHub
2. Import project into Vercel
3. Add your domain in Vercel dashboard
4. Update DNS records as instructed by Vercel

SSL is automatic.
